# 俄罗斯方块 (Tetris) - Windows 桌面游戏规格说明

## 1. 项目概述

- **项目名称**: 俄罗斯方块 (Tetris)
- **类型**: 2D 桌面益智游戏
- **核心功能**: 经典俄罗斯方块玩法，支持移动、旋转、加速下落、自动消行
- **目标平台**: Windows 桌面

## 2. 技术栈

- **语言**: Python 3.x
- **图形库**: Pygame 2.x
- **打包工具**: PyInstaller (生成独立 .exe 文件)

## 3. 游戏界面规格

### 3.1 窗口设置
- 窗口大小: 480 x 640 像素
- 背景颜色: #1a1a2e (深蓝色)
- 帧率: 60 FPS

### 3.2 游戏区域
- 主游戏区: 左侧，10列 x 20行
- 每个方块单元: 28 x 28 像素
- 游戏区边框: 2px, #4a4a6a
- 预览区: 右侧，显示下一个方块
- 信息面板: 右侧，显示分数和等级

### 3.3 方块颜色 (7种经典方块)
| 方块类型 | 颜色代码 |
|---------|---------|
| I型 | #00f5ff (青色) |
| O型 | #ffff00 (黄色) |
| T型 | #bf00ff (紫色) |
| S型 | #00ff00 (绿色) |
| Z型 | #ff0000 (红色) |
| J型 | #0000ff (蓝色) |
| L型 | #ff8000 (橙色) |

### 3.4 字体
- 标题/分数: "Courier New", 粗体
- 游戏分数: 24px
- 等级/提示: 18px

## 4. 游戏机制规格

### 4.1 方块生成
- 7种方块 (I, O, T, S, Z, J, L) 随机生成
- "下一个方块"预览显示

### 4.2 控制方式
| 按键 | 功能 |
|-----|------|
| ← / A | 左移 |
| → / D | 右移 |
| ↓ / S | 加速下落 |
| ↑ / W | 旋转方块 |
| 空格 | 硬降 (直接落到底部) |
| P | 暂停/继续 |
| Esc | 退出游戏 |

### 4.3 消行规则
- 填满一行即消除
- 一次消除的行数影响得分:
  - 1行: 100分
  - 2行: 300分
  - 3行: 500分
  - 4行 (Tetris): 800分

### 4.4 等级系统
- 每消除10行升一级
- 初始速度: 1格/500ms
- 每升一级速度提升10%

### 4.5 游戏结束
- 新方块无法放置时游戏结束
- 显示 "GAME OVER" 和最终分数
- 按 R 键重新开始

## 5. 音效 (可选)
- 使用简单的系统蜂鸣声模拟

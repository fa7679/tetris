# 俄罗斯方块 - Windows 桌面游戏

## 运行方式

### 方式一：双击启动
双击 `启动游戏.bat` 文件即可运行游戏。

### 方式二：命令行运行
```bash
python tetris.py
```

## 游戏操作

| 按键 | 功能 |
|-----|------|
| ← / A | 左移方块 |
| → / D | 右移方块 |
| ↓ / S | 加速下落 |
| ↑ / W | 旋转方块 |
| 空格 | 硬降（直接落到底部） |
| P | 暂停/继续 |
| R | 重新开始（游戏结束后） |
| Esc | 退出游戏 |

## 游戏规则

- 填满一行即消除
- 得分规则：
  - 1行：100分 × 等级
  - 2行：300分 × 等级
  - 3行：500分 × 等级
  - 4行（Tetris）：800分 × 等级
- 每消除10行升一级
- 等级越高，方块下落速度越快

## 系统要求

- Python 3.x（内置 tkinter 库，无需额外安装依赖）
